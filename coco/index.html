            Converts .cas (Cassette) data files to .wav to play into a real Color Computer
            Works well depnding on sound hardware..  Seems to work well on most PCs audio out with sound on very high without distortion..
            If it don't work on one, try anouther.  Does not seem to work on my Samsung Phones/Tablets.. Maybe sound is not loud enough?
            Currently no options or validation to keep it as simple as posible..  only supports 9600 bit rate.
            Not sure other rates would help for other audio hardware? 
            23-03-15 v1.1 - Added Unzip You can have multiple files in a zip and will list them. Double click to Load.
            23-03-21 v1.2 - Added Timex/Sinclair ZX81.. .P files..  Yes, my first computer and have one so...
            23-03-23 v1.3 - Added SHA-1 Checksums.. Can do 256/512 but overkill.. Used to compare files for dups or diffs, wav diffs, debugging.
            23-03-29 v1.4 - Added Coco .DSK files.. Not perfect and does not work with all dsk formats.  Only simple 1 game .DSKs which seem to be most common.
